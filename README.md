# ML Summer 2026

Минималистичный статический HTML-дашборд для отслеживания прогресса в изучении Data Science и Machine Learning летом 2026 года.

Цель проекта — видеть общий прогресс от новичка до уровня стажера / Junior ML Engineer: учебные блоки, темы, проекты, Kaggle-задачи, навыки и ежедневную активность.

## Как открыть

Откройте файл `index.html` в браузере двойным кликом или через контекстное меню.

Backend, сборка и установка зависимостей не нужны. Дашборд использует только HTML, CSS и JavaScript.

## Возможности

- общий процент прогресса;
- счетчик выполненных тем;
- streak дней обучения подряд;
- 8 учебных блоков Roadmap;
- чекбоксы, статусы, заметки и даты завершения для каждой темы;
- дневной трекер с историей последних записей;
- трекер проектов с этапами выполнения;
- шкала навыков от 0 до 5;
- переключение light/dark mode;
- экспорт, импорт и сброс прогресса.

## Хранение данных

Все данные сохраняются в `localStorage` браузера. Это значит, что прогресс остается на этом же устройстве и в этом же браузере после закрытия страницы.

## Экспорт и импорт

Нажмите `Export progress`, чтобы скачать JSON-файл с текущим прогрессом.

Нажмите `Import progress`, выберите ранее экспортированный JSON-файл, и дашборд восстановит сохраненное состояние.

Кнопка `Reset progress` полностью сбрасывает данные после подтверждения.
